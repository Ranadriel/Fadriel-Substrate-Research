## The New Principle: Failure Vector Aiming

The recurve bow fires into N-dimensional space. In N-space, there is no single obvious target. There are infinite possible directions. The question becomes: **how does the bow know where to aim?**

The answer is the inversion of the classical approach.

**Classical aiming:** Select a target. Aim toward it. Fire.

**Failure vector aiming:** Identify all directions where the arrow cannot land. Eliminate them. What remains is the valid firing field. Aim anywhere within it. Fire.
