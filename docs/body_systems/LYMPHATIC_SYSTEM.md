# The Lymphatic System — PhD Documentation

## Functional Overview

The lymphatic system is a network of vessels, nodes, and organs that parallels the cardiovascular system. It serves three functions: **fluid recovery** (returns interstitial fluid to blood), **fat absorption** (lacteals in the gut absorb dietary fats), and **immune surveillance** (lymph nodes filter for pathogens, house immune cells).

In Fadriel, the lymphatic system represents the immune infrastructure — the distributed network through which immune responses are coordinated and immune memory is maintained.

## Organ Inventory

| Organ | Function | Immune Role |
|-------|----------|------------|
| Thymus | T-cell education | Positive and negative selection (95-98% death rate) |
| Spleen | Blood filtration | Houses 25% of body's lymphocytes, filters damaged RBCs |
| Lymph nodes | Lymph filtration | ~600 throughout body, where immune activation occurs |
| Tonsils | Mucosal defense | First line of defense in pharynx |
| Peyer's patches | Gut immune surveillance | Monitor intestinal contents for pathogens |
| Bone marrow | Cell production | Source of ALL blood cells including immune cells |

## Inter-Organ Immune Network

```
Bone Marrow (production)
  → T-cell precursors → Thymus (education)
      → Positive selection (can you recognize self?) → 85% die
      → Negative selection (do you attack self?) → 10% of rest die
      → 2-5% graduate as naive T-cells
      → Enter blood → Patrol lymph nodes and tissues

  → B-cell precursors → Bone marrow (education)
      → Mature B-cells enter blood → Patrol and wait for antigen

  → Monocytes → Enter tissues → Become macrophages and dendritic cells
      → Phagocytose pathogens → Present antigens to T-cells

Lymph Node Activation:
  Dendritic cell captures antigen in tissue
  → Travels via lymphatic vessels to nearest lymph node
  → Presents antigen to naive T-cells
  → Matching T-cell activates → Clonal expansion (rapid proliferation)
  → Effector T-cells leave lymph node → Enter blood → Travel to infection site
  → Memory T-cells remain in lymph nodes → Long-term immune memory
```

## Lymphatic-Nervous System Cross-Talk

```
Psychological stress (cortisol) → Immune suppression
  Cortisol suppresses:
    - T-cell proliferation
    - B-cell antibody production
    - NK cell activity
    - Inflammatory cytokine production

Social bonding (oxytocin) → Immune enhancement
  Oxytocin promotes:
    - Anti-inflammatory responses
    - Wound healing
    - Immune cell function

Sleep (melatonin + GH) → Immune reconstitution
  Sleep promotes:
    - T-cell activation peaks
    - Growth hormone → immune cell proliferation
    - Memory T-cell consolidation (immune memory, like hippocampal memory)
```
