Zest syntax centers around 5 core verbs acting upon physical/esoteric interaction laws.

3. **Couple** - Bond two structures. If one absorbs energy, it transmits to its coupled pair.
   `couple hippocampus with visual_cortex`
4. **Pulse** - The "Heartbeat". This propagates changes across the entire bonded GPU graph matrix per tick.
   `pulse brain`
5. **Emit** - Extract crystallized logic dynamically into local interfaces, shells, or photons (visual boundaries).
   `emit compiler -> boundary.binary`
