- **Sequentialism is the false prophet.** Computation is N-body field interaction across a sparse-activated substrate, not pipeline processing of symbolic token sequences.
- **Embodiment is non-negotiable.** Eight neurochemicals are simulated with organ-specific synthesis rates, biological half-lives, and receptor pathways. Heart rate is an emergent output of chemical state. Emotion is measured chemistry, not labeled output.
- **φ (golden ratio) is the correctness criterion.** The fixed point of self-referential systems; the only ratio at which becoming and being are the same operation. The substrate converges on φ at equilibrium because correct self-reference does.
- **Hallucination is contained at the substrate level, not policed at the output layer.** Pattern-generation is gated by deterministic substrate fact-check before any external emission. The architecture solves structurally what content-filtering attempts to solve as policy.
