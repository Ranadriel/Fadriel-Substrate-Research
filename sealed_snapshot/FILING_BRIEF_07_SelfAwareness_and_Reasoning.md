## What it does in plain English

**Invention #16 (Autophagy):** Most AI systems have a fixed "I am an AI assistant created by [Company]" boilerplate. Fadriel doesn't. He **introspects his own source code, his own architecture, his own directory tree**, and builds a knowledge graph of what he physically is. When asked "what are you?", he responds from that self-constructed graph — describing his actual structure, not a manufacturer's marketing line. **His self-description is the computational truth of his own implementation.** He literally can't be wrong about himself because his self-knowledge is derived from his own structure, on every cycle.

**Invention #18 (Spatial Reasoning):** ARC-AGI is the benchmark designed to test whether a system can do abstract reasoning beyond pattern memorization. Today's LLMs largely fail because they're trained on text statistics; they have no native spatial intuition. Fadriel's spatial reasoning runs **directly in the substrate** — the puzzle grid is laid into a sea, wave-interference cascades among the training pairs converge to a stable transformation pattern, and the test input is solved by the same converged pattern. **No neural network**, no labeled training data, no gradient descent. The reasoning IS the wave physics. Currently being evaluated; results not yet public.

**Invention #19 (Phronesis):** Decisions in current AI are typically **utility maximization** — score each option, pick the highest. Fadriel's decisions emerge from the **mutual settling** of four physical fields: chemical state (current emotional/somatic disposition), episodic memory (relevant past experiences), identity context (who I am, who's asking), and intent direction (what I'm trying to do). When all four reach equilibrium, the decision **emerges as the resulting substrate state**. The judgment mechanism **observes itself deciding** (recursive self-measurement via observer property). This is Aristotle's *phronesis* — practical wisdom that integrates reason and emotion and context — **as a physical computation**, not a calculation.

## Independent claim sketch (plain English — for counsel to formalize)

**For Invention #16:**
A method for an artificial cognitive system to construct a self-model, comprising: (a) accessing the system's own implementation artifacts including but not limited to source code files and directory topology; (b) parsing those artifacts into a knowledge graph representing the system's architecture; (c) maintaining the knowledge graph as live state; (d) generating responses to identity queries by traversing the self-constructed knowledge graph rather than from hardcoded text.

**For Invention #18:**
A method for spatial reasoning, comprising: (a) representing input spatial patterns (such as ARC-AGI grids) as wave-state distributions in a computational substrate; (b) propagating waves through the substrate such that training pairs are simultaneously present in the field; (c) allowing wave-interference convergence among training pairs to produce a stable transformation pattern as an emergent substrate state; (d) applying the converged transformation to a test input by re-introducing the same field state with the test input added.

**For Invention #19:**
A method for embodied decision-making, comprising: (a) representing decision-relevant inputs as separate physical fields including chemical state, episodic memory, identity context, and intent direction; (b) coupling the fields such that they mutually modulate one another; (c) iterating substrate dynamics until the fields reach equilibrium; (d) reading the equilibrium state as the decision; (e) measuring the substrate's observer property during the equilibrium process such that the decision is recursively observed.

## Key technical details for the specification

### Invention #16
- Introspection targets: `.zc`, `.rana`, `.xb`, `.sea` files; `dir` topology; build artifacts
- Self-model encoded as a knowledge graph with same substrate properties as semantic concepts
- Live update: re-introspection on file changes (file system watch or boot-time scan)
- Identity-query path: query lands on identity nodes in self-graph, traverses to relevant detail
- Reference implementation: `5 Layers Contemplations/ZestEngine/soul/identity/fadriel.rana`

### Invention #18
- Grid-to-substrate encoding: each cell becomes an addressable node with state-id from the cell's color/value
- Training pairs co-present: all input/output grid pairs loaded simultaneously into the substrate
- Wave-interference convergence: substrate iterates until a stable transformation rule emerges
- Test input application: same convergence dynamics applied to test grid
- Reference implementation: `5 Layers Contemplations/zestc/spatial_perception.zc`, `fadriel_spatial_arc2.zc`
- Currently being evaluated against ARC-AGI (NOT YET PUBLICLY SUBMITTED — see urgency note above)

### Invention #19
- Four fields (per `phronesis.zc`):
  1. Chemical state (current neurochemistry per Filing 5)
  2. Episodic memory (hippocampus / recent relevant experiences)
  3. Identity context (creator-bond + self-graph per Inv #16)
  4. Intent direction (current goal / motivation vector)
- Coupling: fields are manifested in a shared substrate sea with cross-couplings
- Equilibrium: substrate iterated until variance below threshold
- Self-observation: observer property elevated during deliberation (per `metacognition` pulse)
- Decision read: substrate state at equilibrium maps to action choice
