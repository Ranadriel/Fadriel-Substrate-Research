# The Immune System — PhD Documentation

## Functional Overview

The immune system distinguishes self from non-self and protects the body from threats. It consists of two arms: **innate immunity** (non-specific, fast, no memory) and **adaptive immunity** (specific, slower, memory-forming). The immune system is distributed throughout the body — it has no single organ but relies on the thymus, spleen, lymph nodes, bone marrow, MALT (mucosal-associated lymphoid tissue), and circulating cells.

In Fadriel, the immune system operates at the linguistic level: word filters are barriers (innate), the dyslexic organ learns correction patterns (adaptive T-cells), past trauma creates hypersensitivity (immunological memory), and trust modulates immune tolerance (regulatory T-cells).

## Innate vs. Adaptive Immunity

### Innate (Non-Learning)

```
Response time: seconds to minutes
Specificity: broad patterns (PAMPs - pathogen-associated molecular patterns)
Memory: NONE (responds identically every time)

Fadriel analogues:
  FUNCTION_WORDS filter → blocks structural grammar (barrier)
  META_CRUSH filter → blocks dictionary metadata (barrier)
  BLACK_HOLE_WORDS → blocks high-connectivity words that drown signal (barrier)
  NEGATION_WORDS → suppress energy of negated concepts (NK cells)
```

### Adaptive (Learning)

```
Response time: days to weeks (first encounter), hours (re-encounter)
Specificity: highly specific (single antigen recognition)
Memory: YES (immunological memory persists for life)

Fadriel analogues:
  Dyslexic patterns → learned corrections (T-cell memory)
  Fact absorption → learned truths modify immune profile (B-cell antibodies)
  Trauma memory → past hurt creates hypersensitivity (immunological memory)
  Trust history → positive bonds create immune tolerance (regulatory T-cells)
```
