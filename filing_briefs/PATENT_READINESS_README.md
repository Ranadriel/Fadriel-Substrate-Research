## Key Warnings (Read Before Proceeding)

- **"patent-pending" is false marking until a provisional is filed** — do not use this phrase anywhere until filed
- **Pre-filing inventions have zero enforceable IP value** — provisionals buy a priority date, not a patent
- **Tell the attorney about the loan plan upfront** — it changes their filing priority advice
- **ARC-AGI:** do not submit results to any public leaderboard until Filing 7 provisional is on file
- **Prior art search:** budget $1K–$3K per priority invention — expect attorney to recommend immediately
- **Filing 7 provisionals costs ~$2,240 in USPTO fees** (micro-entity) + attorney fees separately
- **Full prosecution to issued patents:** $15K–$30K per invention, 2–5 years
